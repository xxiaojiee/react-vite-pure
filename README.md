# React

### 基于 Vite + React + TypeScript 的脚手架，开箱即用

> 快速开始React + ts 的项目, 5秒极速启动 VS webpack 20 秒

## 技术栈

* React17
* TypeScript
* React-Router
* Vite 2

## 快速开始

```bash
git clone https://github.com/YanYuanFE/vite-react-app
```

进入目录安装依赖:

```bash
cd vite-react-app
#国内用户推荐yarn或者cnpm
npm install 或者 yarn install
```

开发：

```bash
npm run dev
```

构建：

```bash
npm run build
```
